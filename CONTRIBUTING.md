# Contributing Guide

## Workflow

1. Open Ubuntu
2. Go to the repo
3. Run git pull
4. Open VS Code using code .
5. Reopen in Dev Container if prompted
6. Work on your assigned files
7. Run git status
8. Run git add .
9. Run git commit -m "Describe your changes"
10. Run git push

## Team Rules

- Pull before starting work
- Use clear commit messages
- Do not edit another member's files without asking
- Keep documentation updated
- Put code in the correct subsystem folder
